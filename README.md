# TRINETRA - AI Stock Decision Assistant

A next-generation AI-powered stock decision assistant that provides clear, actionable market insights using advanced LLM integration.

## Features
- Real-time market data analysis.
- AI-driven decision support for stock trading.
- Clean, responsive UI built with React and Tailwind CSS.

## Getting Started
1. Install dependencies: `npm install`
2. Set your `GEMINI_API_KEY` in `.env.local`
3. Run the app: `npm run dev`
